# ZeroicAI Twitter Bot 

Autonomous Twitter bot posting bull content about AI, agents, crypto, and meme coins.

## Features

-  **Automated Posting** - 4 tweets per day (every 6 hours)
-  **5 Content Categories**:
  - AI & Machine Learning
  - ZeroicAI Framework
  - Crypto & DeFi
  - Meme Coins
  - General Bull Posts
-  **Content Filtering** - Safety checks before posting
-  **FREE Tier** - Uses X's free API (500 posts/month)

## Bot Behavior

- **Frequency**: 4 tweets per day (every 6 hours)
- **Categories**: Random selection from 5 categories
- **Signature**: All tweets marked with 🤖 
- **Safety**: Content filter validates before posting
